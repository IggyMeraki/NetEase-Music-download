先安装 文件所需要的依赖模块 
pip install -r requirements.txt
再运行main.py文件即可

## 环境要求
Python >= 3

## 使用说明书

- 运行main.py或release中的exe文件，会自动启动网页


## 音质说明

- 特定音质只有有会员的时候才能下载，所以做了选项的区分，请务必确认登录的账号的VIP状态！
- 在cookie失效或没有cookie的时候会触发登录页面，在弹出的网页中登录即可。
- 音质越高所需要的时间更长，请耐心等待
